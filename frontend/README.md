# FinSight AI Frontend

React-based frontend for FinSight AI personal finance platform.

## Features

- Modern, responsive dashboard
- Transaction management
- Category management
- Analytics and visualizations
- Goal tracking
- Financial health score display

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Recharts
- Zustand (state management)
- React Router
- Axios
